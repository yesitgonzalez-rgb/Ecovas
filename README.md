# Tu Tienda de la Salud — Sitio Web

Sitio estático (HTML/CSS/JS puro, sin dependencias de build) listo para publicar en cualquier hosting.

## Estructura
- `index.html` — página completa
- `assets/css/style.css` — estilos
- `assets/js/script.js` — slider, menú móvil, animaciones
- `assets/video/hero.mp4` — **agrega aquí tu video real del home** (formato mp4, recomendado <15MB, 1920x1080 o 1280x720)
- `assets/img/hero-poster.jpg` — imagen de respaldo mientras carga el video
- `assets/img/` — agrega aquí fotos reales de productos si quieres reemplazar los íconos

## Cómo verlo
Abre `index.html` directamente en el navegador, o sirve la carpeta con cualquier servidor estático.
